# 截图强国

学习强国截图生成器

## Getting Started

- 适用范围:适用于盛行截图主义的党支部
- 适用机型:
    - iPhone x
    - iPhone xr
    - iPhone xs
    - iPhone xs max
- 适用人群:积分大于100分小于100000分的人群(100分一下的分数可能会出现偏移, 不在正中间)
- 使用方法: 
    - 配置安装flutter sdk
    - 项目下运行`flutter run`
